# 연구자 맞춤형 다각적 AI 학술자료 추천 시스템

## 사용 모델

### 1. 임베딩 모델
- OpenAI `text-embedding-3-small`
- 사전 학습된 모델이며 로컬 가중치 파일은 존재하지 않음
- 임베딩은 API 호출을 통해 생성

### 2. sLLM 모델
- OpenAI `gpt-4o-mini`
- 사전 학습된 모델이며 로컬 가중치 파일은 존재하지 않음
- 쿼리, 선정 사유 생성에 API 호출 방식 사용

## 사용 방법
- OpenAI API 키 필요
   ```bash
   OPENAI_API_KEY="your_api_key_here"
   ```
